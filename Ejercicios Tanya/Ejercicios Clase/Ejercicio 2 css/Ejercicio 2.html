Conceptos en lista descriptiva

  • Function: Un bloque de código que realiza una tarea específica y puede ser llamado desde otras partes del programa.
  • let: Una palabra clave para declarar variables con ámbito de bloque, lo que significa que solo existen dentro del bloque en el que se declaran.
  • const: Una palabra clave para declarar constantes, que son valores que no cambian una vez que se les asigna un valor inicial. También tienen ámbito de bloque.
  • Video: La etiqueta video se utiliza para insertar videos en una página web.
  • Canvas: La etiqueta canvas se utiliza para dibujar gráficos y animaciones en tiempo real utilizando JavaScript.
